Frontend Architecture - NoWayOut Migration

User Browser

The frontend architecture consists of a React Single Page Application served globally through AWS CloudFront CDN to reduce latency. It handles localized content for Saudi markets and communicates with backend services through secured API calls.

Backend Architecture - NoWayOut Migration

API Gateway
Auth Service (OAuth, JWT)
Booking Microservice
Payment Gateway Integration
Database (MySQL)
Redis Cache
The backend system is designed with microservices accessed via an API Gateway for scalability. It includes authentication services using OAuth and JWT, booking and payment microservices, with data storage handled by MySQL and caching accelerated by Redis.

Deployment Infrastructure - NoWayOut Migration

AWS Route53
AWS CloudFront (CDN)
AWS EC2 (App Servers)
RDS MySQL
Elasticache Redis
The deployment infrastructure is built on AWS cloud services, with Route53 managing DNS, CloudFront distributing static assets, EC2 instances hosting application servers, and managed RDS and ElastiCache services ensuring highly available and performant data layers.
